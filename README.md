# BCalc

A simple bond calculator to calculate price, duration and DV01

## Installation

    $ git clone https://github.com/tomplace/bcalc
 
## Run 
	$> python bcalc.py

## Help 
	$> python bcalc.py -h

## Roadmap
	This is currently a very bare-bones implementation. Contributions are encouraged!
